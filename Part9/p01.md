# 📘 Chapter 9.1: Natural Language Processing (NLP) with RNN

---

## 🧠 What is NLP?

> **Natural Language Processing (NLP)** is about helping computers **understand, interpret, and generate human language**.

👉 Language = sequences of words
👉 Perfect use case for **RNN**

---

## 🧩 Why RNN is Useful in NLP

Language has:

* Order (word sequence matters)
* Context (meaning depends on previous words)

👉 Example:

* “He went to the bank”
* Meaning depends on context

👉 RNN can:

* Remember previous words
* Use them to understand current word

## 🧠 Core NLP Pipeline with RNN

---

## 🔁 Step 1: Text → Numbers

* Convert words to indices
* Use vocabulary mapping

---

## 🔄 Step 2: Embedding

* Convert words into vectors
* Capture meaning

---

## 🔄 Step 3: RNN Processing

* Sequence passed step-by-step
* Hidden state stores context

---

## 🔄 Step 4: Output Layer

* Predict:

  * class
  * next word
  * sequence

---

## 🏗 Simple Example: Sentiment Analysis

```python id="nlpcode1"
class SentimentRNN(nn.Module):
    def __init__(self):
        super(SentimentRNN, self).__init__()
        
        self.embedding = nn.Embedding(1000, 50)
        self.rnn = nn.RNN(50, 16, batch_first=True)
        self.fc = nn.Linear(16, 2)
    
    def forward(self, x):
        x = self.embedding(x)
        out, _ = self.rnn(x)
        out = out[:, -1, :]
        out = self.fc(out)
        return out
```

---

## 🧠 What This Model Does

* Converts words → vectors
* Processes sequence
* Outputs sentiment

## 📈 Why RNN Works Well in NLP

| Feature           | Benefit                             |
| ----------------- | ----------------------------------- |
| Memory            | Understand context                  |
| Sequence handling | Maintains order                     |
| Flexible length   | Works with different sentence sizes |

## ⚠️ Limitations of RNN in NLP

* Struggles with long sentences
* Slow training (sequential processing)
* Cannot capture very long dependencies

---

## 🚀 Modern Alternatives

Because of limitations, newer models are used:

* LSTM
* GRU
* Transformers (like GPT)

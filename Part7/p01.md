# 📘 Chapter 7.1: Setting Up Environment

---

## 🧠 Why This Step is Important

Before building RNN, LSTM, or GRU models, you need a **working environment**.

👉 Think of it like:

* Before cooking → you arrange kitchen tools
* Before coding → you set up software tools

---

## 🔧 What You Will Install

We will use:

* 🐍 **Python** → main language
* 🔥 **PyTorch** → build neural networks
* 📊 **NumPy** → numerical operations
* 📈 **Matplotlib** → graphs

---

## 🧩 Step 1: Install Python

👉 If not installed:

* Download from: Python official website
* Install latest version (3.x)

---

### ✅ Check Installation

Open terminal / command prompt:

```bash
python --version
```

👉 You should see something like:

```
Python 3.x.x
```

---

## 🧩 Step 2: Install pip (if needed)

Usually comes with Python.

Check:

```bash
pip --version
```

---

## 🧩 Step 3: Create Virtual Environment (Very Important)

👉 Keeps your project clean and isolated

```bash
python -m venv rnn_env
```

---

### Activate Environment

#### Windows:

```bash
rnn_env\Scripts\activate
```

#### Mac/Linux:

```bash
source rnn_env/bin/activate
```

---

## 🧩 Step 4: Install Required Libraries

Now install all needed packages:

```bash
pip install torch numpy matplotlib
```

---

## 🧠 What Each Library Does

| Library    | Purpose                         |
| ---------- | ------------------------------- |
| PyTorch    | Build and train neural networks |
| NumPy      | Work with numbers and arrays    |
| Matplotlib | Plot graphs and results         |

---

## 🧩 Step 5: Verify Installation

Open Python:

```bash
python
```

Then test:

```python
import torch
import numpy as np
import matplotlib.pyplot as plt

print("Setup successful!")
```

👉 If no error → everything is ready

---

## 🧩 Step 6: Optional (Recommended Tools)

### 🧑‍💻 Code Editor

You can use:

* Visual Studio Code
* PyCharm

---

### 📓 Notebook (for easy learning)

Install:

```bash
pip install notebook
```

Run:

```bash
jupyter notebook
```

👉 This opens browser-based coding environment

## ⚠️ Common Issues (Quick Fix)

### ❌ “torch not found”

👉 Solution:

```bash
pip install torch
```

---

### ❌ Environment not activated

👉 You may be installing globally instead of locally

---

### ❌ Version conflicts

👉 Always use virtual environment

## 🔄 Step 1: Prepare Dataset

```python
import torch
import numpy as np

# Sample data
data = [10, 12, 14, 16, 18, 20]

# Create sequences
X = []
y = []

for i in range(len(data) - 3):
    X.append(data[i:i+3])
    y.append(data[i+3])

X = torch.tensor(X, dtype=torch.float32)
y = torch.tensor(y, dtype=torch.float32)

# Add feature dimension
X = X.unsqueeze(-1)
y = y.unsqueeze(-1)
```

---

## 🧠 Understanding This

```id="ts3"
[10,12,14] → 16  
[12,14,16] → 18  
[14,16,18] → 20  
```

👉 Model learns pattern from sequences

---

## 🏗 Step 2: Build RNN Model

```python
import torch.nn as nn

class TimeSeriesRNN(nn.Module):
    def __init__(self):
        super(TimeSeriesRNN, self).__init__()
        
        self.rnn = nn.RNN(
            input_size=1,
            hidden_size=10,
            batch_first=True
        )
        
        self.fc = nn.Linear(10, 1)
    
    def forward(self, x):
        out, _ = self.rnn(x)
        
        out = out[:, -1, :]
        out = self.fc(out)
        
        return out
```

---

## ⚙️ Step 3: Train the Model

```python
model = TimeSeriesRNN()
criterion = nn.MSELoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.01)

for epoch in range(200):
    
    output = model(X)
    loss = criterion(output, y)
    
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
    
    if epoch % 20 == 0:
        print(f"Epoch {epoch}, Loss: {loss.item():.4f}")
```

---

## 🔍 Step 4: Test Prediction

```python
test_seq = torch.tensor([[16, 18, 20]], dtype=torch.float32).unsqueeze(-1)

prediction = model(test_seq)

print("Next value prediction:", prediction.item())
```

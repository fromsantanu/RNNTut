## 🧩 Problem We Will Solve

👉 Predict the **next number in a sequence**

Example:

```
[1, 2, 3] → 4  
[2, 3, 4] → 5  
```

👉 This is a **sequence problem**, perfect for RNN.

---

## 🔧 Step 1: Import Libraries

```python
import torch
import torch.nn as nn
import torch.optim as optim
```

---

## 📦 Step 2: Create Dataset

```python
# Input sequences
X = [
    [1, 2, 3],
    [2, 3, 4],
    [3, 4, 5]
]

# Target outputs
y = [4, 5, 6]

# Convert to tensors
X = torch.tensor(X, dtype=torch.float32)
y = torch.tensor(y, dtype=torch.float32)

# Add feature dimension (important for RNN)
X = X.unsqueeze(-1)   # shape: (samples, sequence_length, input_size)
y = y.unsqueeze(-1)
```

---

## 🧠 Understanding Shape (Very Important)

```
X shape = (3, 3, 1)
```

Meaning:

* 3 samples
* 3 time steps
* 1 feature per step

👉 RNN always expects:

```
(batch_size, sequence_length, input_size)
```

---

## 🏗 Step 3: Define RNN Model

```python
class SimpleRNN(nn.Module):
    def __init__(self):
        super(SimpleRNN, self).__init__()
        
        self.rnn = nn.RNN(
            input_size=1,
            hidden_size=8,
            batch_first=True
        )
        
        self.fc = nn.Linear(8, 1)
    
    def forward(self, x):
        out, _ = self.rnn(x)
        
        # Take last time step output
        out = out[:, -1, :]
        
        out = self.fc(out)
        return out
```

---

## 🧠 What’s Happening Here?

* `nn.RNN` → processes sequence
* `hidden_size=8` → memory size
* `out[:, -1, :]` → last time step
* `Linear` → converts to final prediction

---

## ⚙️ Step 4: Initialize Model

```python
model = SimpleRNN()
criterion = nn.MSELoss()
optimizer = optim.Adam(model.parameters(), lr=0.01)
```

---

## 🔁 Step 5: Training Loop

```python
for epoch in range(200):
    
    # Forward pass
    output = model(X)
    loss = criterion(output, y)
    
    # Backward pass
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
    
    if epoch % 20 == 0:
        print(f"Epoch {epoch}, Loss: {loss.item():.4f}")
```

---

## 🧠 What This Loop Does

Each iteration:

1. Predict output
2. Compare with actual
3. Calculate error
4. Update weights

👉 Same process you learned earlier

---

## 🔍 Step 6: Test the Model

```python
test_seq = torch.tensor([[4, 5, 6]], dtype=torch.float32).unsqueeze(-1)

prediction = model(test_seq)
print("Predicted next value:", prediction.item())
```

---

## 🎯 Expected Output

```
Predicted next value: ~7
```

👉 Not exact, but close (model learns pattern)

---

## 🧠 Simple Intuition

The model learns:

> “Each number increases by 1”

---

## 🧩 Real-Life Analogy

Think of teaching a child:

* Show pattern: 1,2,3 → 4
* Show again: 2,3,4 → 5
* Child learns rule → “+1”

👉 RNN learns similarly

---

## ⚠️ Common Mistakes

### ❌ Wrong shape

👉 Always ensure:

```
(batch, sequence, feature)
```

---

### ❌ Not using last output

👉 Use:

```python
out[:, -1, :]
```

---

### ❌ Learning rate too high

👉 Model may not converge

---

## 📌 Key Takeaways

* RNN processes sequence step-by-step
* Hidden state captures pattern
* Final output used for prediction

# 📘 Additional 1: Installation Guide

---

## 🧠 Goal of This Guide

Set up a clean, reliable environment so you can:

* Run all RNN/LSTM/GRU examples
* Avoid version conflicts
* Reproduce results easily

---

## 🔧 What You’ll Install

* 🐍 **Python** (3.9–3.12 recommended)
* 🔥 **PyTorch**
* 📊 **NumPy**
* 📈 **Matplotlib**
* 📓 **Jupyter Notebook** (optional but useful)

---

## 🧩 Step 1: Install Python

1. Download from the official site: Python
2. During installation:

   * ✔️ Check **“Add Python to PATH”**
   * ✔️ Install `pip` (usually included)

---

### ✅ Verify Installation

```bash
python --version
pip --version
```

You should see versions printed without errors.

---

## 🧩 Step 2: Create a Virtual Environment (Recommended)

👉 Keeps your project isolated (no conflicts with other projects)

```bash
python -m venv rnn_env
```

### Activate

* **Windows**

```bash
rnn_env\Scripts\activate
```

* **Mac/Linux**

```bash
source rnn_env/bin/activate
```

👉 You’ll see `(rnn_env)` in the terminal.

---

## 🧩 Step 3: Install Core Libraries

```bash
pip install torch numpy matplotlib jupyter
```

> This installs **PyTorch**, **NumPy**, **Matplotlib**, and **Jupyter Notebook**.

---

## 🧩 Step 4: (Optional) GPU Support for PyTorch

If you have an NVIDIA GPU, install CUDA-enabled PyTorch from the official selector on PyTorch site.

Example (may vary by system):

```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

👉 If unsure, CPU version is perfectly fine for learning.

---

## 🧩 Step 5: Verify Everything Works

Open Python:

```bash
python
```

Run:

```python
import torch
import numpy as np
import matplotlib.pyplot as plt

print("Torch version:", torch.__version__)
print("NumPy version:", np.__version__)
print("Setup successful!")
```

👉 No errors = good to go.

## ⚠️ Common Issues & Quick Fixes

---

### ❌ `python` not recognized

👉 Reinstall **Python** and enable PATH

---

### ❌ `torch` not found

```bash
pip install torch
```

---

### ❌ Permission errors (Windows)

Run terminal as **Administrator**

---

### ❌ Environment not active

👉 Activate `rnn_env` before installing/running

---

### ❌ Slow installs / network issues

```bash
pip install --upgrade pip
```

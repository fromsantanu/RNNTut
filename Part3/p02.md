# 📘 Chapter 3.2: Activation Functions in RNN

---

## 🧠 Why Do We Need Activation Functions?

From the last chapter, you saw the RNN equation:

* It combines **input + past memory**

But if we stop there:

* The model behaves like a **simple linear system**
* It cannot learn complex patterns

👉 So we need something that can:

> Add **non-linearity** (ability to learn complex relationships)

That “something” is an **activation function**.

---

## 🔑 What is an Activation Function? (Simple Idea)

Think of it like a **decision filter**:

* Input comes in
* It transforms it into a useful output

👉 Example:

* Raw marks → Grade (A, B, C)
* Not just direct mapping, but processed

---

## 🔄 Where Activation is Used in RNN

In the RNN equation:

$h_t = \tanh(W_x x_t + W_h h_{t-1} + b)$

👉 The **tanh** part is the activation function

It decides:

* What to keep
* What to reduce
* What to pass forward

---

## 🟢 Common Activation Functions in RNN

---

## 1️⃣ **tanh (Hyperbolic Tangent)** — Most Common

$y = \tanh(x)$

### 📌 Key Properties:

* Output range: **-1 to +1**
* Centered around zero

---

### 🧠 Why tanh is useful

👉 Keeps values controlled:

* Prevents explosion
* Keeps memory stable

👉 Allows:

* Positive and negative signals

---

### 🧩 Simple Analogy

Think of tanh like a **volume controller**:

* Loud signal → reduced
* Soft signal → amplified

👉 Keeps everything balanced

---

## 2️⃣ **ReLU (Rectified Linear Unit)**

$y = \max(0, x)$

### 📌 Key Properties:

* Negative values → 0
* Positive values → unchanged

---

### 🧠 Why ReLU is used

👉 Very simple and fast
👉 Works well in many networks

---

### ⚠️ Problem in RNN

* Does not limit large values
* Can cause instability in sequences

👉 So less commonly used in basic RNN memory

---

## 3️⃣ **Sigmoid Function**

$y = \frac{1}{1 + e^{-x}}$

### 📌 Key Properties:

* Output range: **0 to 1**

---

### 🧠 Where sigmoid is useful

👉 Acts like a **gate**:

* 0 → block information
* 1 → allow information

👉 Very important in:

* LSTM
* GRU

---

## 🔄 Comparison (Simple View)

| Function | Range   | Use in RNN                  |
| -------- | ------- | --------------------------- |
| tanh     | -1 to 1 | Memory update (most common) |
| ReLU     | 0 to ∞  | Rare in basic RNN           |
| Sigmoid  | 0 to 1  | Gating (advanced RNNs)      |

---

## 🧠 Why tanh is Preferred in Basic RNN

👉 Because it:

* Keeps values small
* Maintains balance
* Handles both positive and negative signals

👉 Which is perfect for **memory representation**

---

## 🔁 What Happens Without Activation?

If no activation is used:

* Model becomes linear
* Cannot learn complex patterns
* Memory becomes weak

👉 Like:

* Trying to understand emotions using only numbers without interpretation

---

## 🧩 Real-Life Analogy

Think of activation like a teacher checking notes:

* Important points → highlighted
* Unimportant → ignored
* Balanced understanding created

---

## ⚠️ Important Limitation (Preview)

* Over many steps, values may shrink
* Memory fades

# 📘 Chapter 3.1: RNN Equation (Basic Understanding)

---

## 🧠 Why Do We Need an Equation?

So far, you’ve understood RNN intuitively:

> It uses **current input + past memory → new memory**

Now we express this idea in a **simple mathematical form**.

👉 Don’t worry—think of this like a **recipe**, not heavy math.

---

## 🔑 The Core RNN Equation

$h_t = \tanh(W_x x_t + W_h h_{t-1} + b)$

---

## 🧩 Breaking It Down (Very Simple)

Let’s decode each part like everyday objects:

---

### 📥 **xₜ (Current Input)**

* What you are seeing **right now**
* Example: current word → “love”

---

### 🧠 **hₜ₋₁ (Previous Memory)**

* What you remember from the past
* Example: previous words → “I”

---

### ⚙️ **Wₓ and Wₕ (Weights)**

* These are like **importance settings**
* They decide:

  * how much to trust current input
  * how much to trust past memory

👉 Example:

* Sometimes present matters more
* Sometimes past matters more

---

### ➕ **b (Bias)**

* A small adjustment value
* Helps fine-tune the result

---

### 🔄 **tanh (Activation Function)**

* Keeps values in a controlled range (between -1 and 1)
* Helps stabilize learning

👉 Think of it like:

* “Don’t let values go out of control”

---

### 📤 **hₜ (New Memory)**

* The updated memory after combining everything
* This is passed to the next step

---

## 🔄 Putting It All Together

At each step, RNN does:

👉 Combine:

* Current input (**xₜ**)
* Previous memory (**hₜ₋₁**)

👉 Adjust using weights

👉 Pass through tanh

👉 Get new memory (**hₜ**)

---

## 🧠 Simple Real-Life Analogy

Think of a student taking notes in class:

* New topic (input)
* Old notes (memory)
* Student decides:

  * what to keep
  * what to update

👉 Updated notebook = new memory (**hₜ**)

---

## 🔁 Step-by-Step Example

Sentence:

```id="eq1"
"I → love → AI"
```

### Step 1:

* Input: “I”
* Memory: empty
* New memory: “I”

---

### Step 2:

* Input: “love”
* Previous memory: “I”
* New memory: “I love”

---

### Step 3:

* Input: “AI”
* Previous memory: “I love”
* New memory: “I love AI”

---

👉 Same equation applied every time

---

## 📦 What About Output?

Sometimes we also compute output:

👉 Conceptually:

* Output depends on current memory (**hₜ**)

Example:

* Predict next word
* Classify sentiment

## 🔁 Important Insight

> The equation is applied **again and again at every time step**

👉 Same formula
👉 Same weights
👉 Different inputs and memory

---

## ⚠️ Why This Simple Equation Has Limits

This equation:

* Works well for short sequences
* But struggles with long memory

👉 Why?

* Information may fade over time

(We’ll study this in **Vanishing Gradient Problem**)

---

## 📌 One-Line Summary

> The RNN equation combines **current input and previous memory using weights and an activation function to produce new memory**.

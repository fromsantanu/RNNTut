# 📘 Chapter 1.1: What is Sequential Data?

---

## 🧠 Simple Idea

**Sequential data** is any data where **order matters**.

👉 If you change the order, the meaning or result changes.

---

## 🔑 Key Characteristics of Sequential Data

### 1. **Order Matters**

* Position of each element is important
* Changing order changes meaning

---

### 2. **Context is Important**

* Current data depends on previous data

👉 Example:

* Word: “bank”
* Meaning depends on earlier words:

  * “river bank”
  * “money bank”

---

### 3. **Data Comes Step-by-Step**

* Not all at once
* Comes in a sequence over time

---

## 🔄 Sequential vs Non-Sequential Data

| Type           | Example                | Order Important? |
| -------------- | ---------------------- | ---------------- |
| Sequential     | Sentence, stock prices | ✅ Yes            |
| Non-sequential | Image, static data     | ❌ No             |

👉 Example:

* In a photo, swapping pixels usually doesn’t change meaning much
* In a sentence, swapping words completely changes meaning

---

## 🧩 Why This Matters in AI

Traditional models (like simple neural networks):

* Treat inputs **independently**
* Do not remember past inputs

But real-world problems:

* Need **memory**
* Need **context**

👉 This is why we need models like **RNN**

---

## 🧠 Intuition Summary

Think of sequential data like:

> 📖 Reading a story — each line depends on the previous one
> 🎬 Watching a movie — scenes make sense only in order

---

## 📌 One-Line Definition

> Sequential data is data where **the order of elements carries meaning and must be preserved**.

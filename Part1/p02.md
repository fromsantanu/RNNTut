# 📘 Chapter 1.2: Why Normal Neural Networks Fail

---

## 🧠 Quick Reminder

From the previous chapter:

> Sequential data = **order matters + past affects present**

Now the key question:

👉 **Can a normal neural network (ANN) handle this?**
👉 Short answer: **No — and here’s why.**

---

## 🧩 How a Normal Neural Network (ANN) Works

A basic ANN processes input like this:

```
Input → Hidden Layers → Output
```

👉 Important point:

* It treats **each input independently**
* It has **no memory of previous inputs**

---

## 📌 Problem 1: No Memory

### 📝 Example: Understanding a Sentence

![Image](https://images.openai.com/static-rsc-4/CJpIPx9B1M6CvHDdJVg_U_N2hLtJ1FKNzc--k-r_49oDHiKQbh18-TL21WGmPqbD4A10-7PFzcGQ8lsjFtGK6jNCivhQlMpsFFAabiIWFDNJuWXyZU9Z8IgRqdSDCYhliEgAQzGcbC2_unojuI5bFb1DMS3X4P7e-T6mq-BTn5JLoGK7XtC1uRFi3NZvjwmj?purpose=fullsize)

Sentence:

> “I went to the bank to deposit money”

To understand **“bank”**, you need earlier words:

* “deposit money” → financial bank

👉 But ANN:

* Looks at “bank” **alone**
* Does not remember previous words

❌ Result: Wrong or incomplete understanding

---

## 📌 Problem 2: Cannot Handle Order

### 🔄 Example: Same Words, Different Meaning

* “Dog bites man”
* “Man bites dog”

👉 Same words, different meaning because of order

But ANN:

* Treats both as **same set of inputs**

❌ Result: Cannot distinguish meaning properly

---

## 📌 Problem 3: Fixed Input Size

### 📏 Example: Sentences of Different Lengths

* “I am happy” (3 words)
* “I am very happy today” (6 words)

ANN expects:

* Fixed number of inputs

👉 Problem:

* Real-world sequences are **variable length**

❌ Result: Not flexible

---

## 📌 Problem 4: No Context Awareness

### 🎙 Example: Speech Recognition

![Image](https://images.openai.com/static-rsc-4/FEJ-BxfiPM8Iz1obXrtg2WPmKY6O-ZHqZ_WGvOemuVQ58dZyrv1WbIwEfC4gCbXgS4ipMnpy-32q1d6xqsiERLauRbczLUFlCtmhC9XPdaqrolKiAa1jJhoU3nOZdbBENPtr35-PpdZjBeST4fpWW3Lhql1l59DtS-gVGE2yetbxsMSdWu3SwqtCnQmKv-Vh?purpose=fullsize)

Speech is:

* Continuous
* Depends on previous sounds

ANN:

* Processes chunks separately
* Cannot connect them

❌ Result: Broken understanding

---

## 📌 Problem 5: Time Dependency Ignored

### 📈 Example: Stock Prices

* Day 1 → ₹100
* Day 2 → ₹105
* Day 3 → ₹110

👉 Future depends on past trend

ANN:

* Sees values independently
* Does not track progression

❌ Result: Poor prediction

---

## 🔄 Summary of Limitations

| Problem           | Why ANN Fails                 |
| ----------------- | ----------------------------- |
| No memory         | Cannot remember past inputs   |
| Ignores order     | Treats data as unordered      |
| Fixed input size  | Cannot handle varying lengths |
| No context        | Misses relationships          |
| No time awareness | Cannot track sequences        |

---

## 🧠 Simple Analogy

Imagine a student who:

* Reads each word separately
* Forgets previous words immediately

👉 Can they understand a paragraph?

❌ No

That’s exactly how ANN behaves with sequential data.

---

## 🎯 So What Do We Need?

We need a model that:

* Remembers past inputs
* Understands order
* Handles sequences of any length
* Uses context

👉 This leads to:

# 🔁 Recurrent Neural Networks (RNN)

---

## 📌 One-Line Insight

> Normal neural networks fail because they **cannot remember or use past information in sequential data**.

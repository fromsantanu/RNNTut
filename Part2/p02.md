# 📘 Chapter 2.2: How RNN Works Internally

---

## 🧠 Quick Recap

From the last chapter:

> RNN = a model that **remembers past information (hidden state)** and uses it while processing new input.

Now let’s open the “black box” and see **what actually happens inside**—in a simple way.

---

## 🔁 The Core Working Idea

At every step, an RNN does three things:

1. **Takes the current input**
2. **Combines it with past memory (hidden state)**
3. **Produces output and updates memory**

👉 This cycle repeats for each step in the sequence.

---

## 🔄 Step-by-Step Flow

### 🧩 Think of it like a loop:

```id="rnnflow1"
Previous Memory + Current Input → New Memory → Output
```

👉 Then:

* New memory becomes the **previous memory** for the next step

---

## 📦 Components Inside an RNN

### 1. Input (xₜ)

* The current data at time step **t**
* Example: a word in a sentence

---

### 2. Hidden State (hₜ)

* The **memory** of the network
* Carries information from previous steps

---

### 3. Output (yₜ)

* The prediction at that step
* Example: next word, class label, etc.

---

## 🔁 Full Internal Flow

At time step **t**, the RNN performs:

* Takes:

  * current input (**xₜ**)
  * previous memory (**hₜ₋₁**)

* Produces:

  * new memory (**hₜ**)
  * output (**yₜ**)

---

## 🧠 The Key Formula (Simple Understanding)

h_t = \tanh(W_x x_t + W_h h_{t-1} + b)

👉 What this means in simple words:

* Combine:

  * current input (**xₜ**)
  * previous memory (**hₜ₋₁**)
* Apply a transformation (tanh)
* Get new memory (**hₜ**)

## 🔁 Important Concept: Same Weights Reused

Unlike normal networks:

* RNN uses **same parameters at every step**

👉 Why?

* Because the task is the same at each step:

  > “Understand current input using past memory”

---

## 🧩 Step-by-Step Example

Sentence:

```id="rnnex1"
"I → love → AI"
```

### Step 1:

* Input: “I”
* Previous memory: empty
* New memory: stores idea of “I”

---

### Step 2:

* Input: “love”
* Previous memory: “I”
* New memory: “I love”

---

### Step 3:

* Input: “AI”
* Previous memory: “I love”
* New memory: “I love AI”

---

👉 Final understanding builds gradually

---

## 🧠 Real-Life Analogy

Think of writing notes while listening to a lecture:

* You hear something new (input)
* You combine it with your notes (memory)
* You update your notes

👉 That notebook = hidden state

---

## 🔄 Types of RNN Outputs (Important)

RNN can work in different ways:

### 1. One-to-One

* Like normal ANN

---

### 2. One-to-Many

* Input → sequence output
* Example: image → caption

---

### 3. Many-to-One

* Sequence → single output
* Example: sentiment analysis

---

### 4. Many-to-Many

* Sequence → sequence
* Example: translation

---

## 📌 Why This Design is Powerful

Because it:

* Keeps memory across steps
* Handles sequences naturally
* Works with variable-length input
